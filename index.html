Standar Keselamatan Kerja Kami

Kami mengutamakan keselamatan dalam setiap pekerjaan untuk melindungi teknisi dan pelanggan

APD Lengkap

Teknisi kami selalu menggunakan Alat Pelindung Diri (APD) lengkap termasuk helm, sarung tangan, dan sepatu safety.

Prosedur Listrik Aman

Selalu mematikan sumber listrik sebelum bekerja dan menggunakan alat ukur untuk memastikan tidak ada arus listrik.

Alat Pemadam Kebakaran

Membawa APAR (Alat Pemadam Api Ringan) untuk antisipasi keadaan darurat saat bekerja dengan listrik atau bahan mudah terbakar.

Penanganan Freon Aman

Mengikuti prosedur penanganan freon yang aman untuk mencegah kebocoran dan kontaminasi lingkungan.

Alat Standar Nasional

Menggunakan peralatan kerja yang memenuhi standar nasional dan dirawat secara berkala.

Asuransi Teknisi

Semua teknisi kami dilindungi asuransi kecelakaan kerja untuk memberikan perlindungan maksimal.
